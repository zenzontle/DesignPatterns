Here's the recommended way to learn them, taken from http://programmers.stackexchange.com/questions/45044/recommended-order-for-learning-design-patterns:
# First
- Factory Method
- Builder
- Composite

# Second
- Prototype
- Observer
- Facade
- Bridge

# Third
- Abstract Factory
- Proxy
- Chain of responsibility
- Visitor

# Fourth
- Singleton
- Command
- State
- Strategy

# Extra
- Adapter
- Decorator
- Flyweight
- Mediator
- Iterator
- Memento
- Template Method
- Interpreter
